# Versioning rules:
    Unstable -> add a letter (example - v3.1.1 -> v3.1.1a)    
    Stable -> Incriment one (example - v3.1.1 -> v3.1.2)    
    Minor update -> Incriment one (example - v3.4.6 -> v3.5)    
    Major update -> Incriment one (example - v3.4.2 -> v4.0)    
    
# Run 
    node index.js
